Nowadays coding is an essential skill that lets us talk to computers. In the past there were low-level programming languages like assembly, C but over time we have developed new languages in order to increase understanding of that code structure and Programming languages have been separated into two categories, low-level and high-level.


Low level language basically means the written is understandable by a machine while it is not that possible to understand it from a human. But high level programming languages provide much more features that help humans to understand the code while the code needs some process to make it understandable.


Both coding scenarios are developed in English syntax and English grammar which is not easy to a non-english-speaker.  

For a new learner who has never heard about the terms and who can’t speak English, it is difficult to understand an English text so English code.


 In that case we may develop new native programming languages to learn programming fast and to make it easy for everyone since as developers we want to understand code. 


As a native-turkish-speaker I have developed a programming language that is written in Turkish grammar and Turkish word-order. Since Turkish is an agglutinative language there are so many suffixes that we put them at the end of verbs which needed to be added to the programming language .


Eventually there was a programming language that could be understood by a Turkish native.

From now on a Turkish speaker can write codes while keeping using their own native language.


The high-school students were being teaching English-related-programming-languages such as Python or C++ and asked to the teachers to teach their student in their own languages -in Turkish- and the teacher taught their students Hüma Language and suddenly students start to understand code easier and faster than other languages.
